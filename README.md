# Infinite Looping Web Ad

A simple, elegant web-based advertisement that loops infinitely with smooth animations and a clean interface.

## Features

- ✨ **Infinite Loop**: Ad plays continuously without stopping
- 🎨 **Smooth Animations**: Fade-in, slide-in, and bounce effects
- 📊 **Loop Counter**: Tracks how many times the ad has cycled
- 🎯 **Call-to-Action Button**: Interactive button for user engagement
- 📱 **Responsive Design**: Works seamlessly on desktop and mobile
- 🎬 **Full-Screen Display**: Takes up the entire viewport

## Files

- **index.html** - Main HTML structure
- **styles.css** - Styling and CSS animations
- **script.js** - JavaScript logic for loop management
- **README.md** - Documentation

## How It Works

1. The ad container cycles every 5 seconds
2. Each cycle triggers smooth animations
3. A loop counter displays how many times the ad has cycled
4. The CTA button is interactive and can trigger custom actions

## Configuration

To customize the ad loop duration, edit `script.js`:

```javascript
this.loopDuration = 5000; // Change this value (in milliseconds)
```

## Usage

1. Open `index.html` in a web browser
2. The ad will start playing and looping infinitely
3. Watch the loop counter increment in the bottom-right corner

## Customization Ideas

- Change the gradient background in `styles.css`
- Update the ad content in `index.html`
- Modify animation timings in the CSS
- Add tracking/analytics in `script.js`
- Add dynamic content loading from an API
